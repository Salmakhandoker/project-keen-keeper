
# 🚀 KeenKeeper

## 📌 Description

KeenKeeper is a modern relationship management web app that helps you keep track of your friends, interactions, and communication history. It allows users to monitor connections, view timelines, and stay socially organized.

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 Tailwind CSS
* ⚡ Vite
* 📦 JavaScript (ES6+)

---

## ✨ Features

* 👥 Add and manage friends
* 📅 Track interaction timeline (Call, Text, Video)
* 📊 Dashboard with real-time stats
* 🚦 Status indicators (On Track, Overdue, Almost Due)
* 🔍 Filter interactions by type
* 🎯 Clean and responsive UI

---

## 📦 Installation

```bash
git clone https://github.com/salma-khandoker/keen-keeper.git
cd keen-keeper
npm install
npm run dev
```
